1. What problem does auto-fit solve?
Auto-fit adjusts the number of columns based on available space.

2. Why is minmax important?
Minmax prevents items from becoming too small.

3. Do items shrink too much?
No, minmax sets a minimum width.

4. Is this layout mobile friendly?
Yes, it adjusts automatically on different screen sizes.

5. Why are media queries not needed here?
The grid responds automatically using auto-fit.
