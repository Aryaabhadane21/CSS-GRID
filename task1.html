1. Difference between a normal div and a grid container:
A normal div does not control the layout of its children, while a grid container can arrange its child elements in rows and columns.

2. Why does the layout look the same after applying Grid?
The layout looks the same because no rows or columns are defined, so the items stay in the default flow.

3. Are grid items created automatically?
Yes, all direct child elements of a grid container automatically become grid items.

4. Can Grid work without rows or columns?
Yes, Grid can still work, but no visible layout change happens until rows or columns are defined.

5. What does Grid need to show visible changes?
Grid needs defined rows, columns, or gaps to visibly change the layout.
